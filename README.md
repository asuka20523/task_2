# 実績ログ

日々のEC運用タスクや実績を記録し、半期評価のときにサッと振り返れるようにするための、シングルファイルのWebアプリです。

## 特徴

- **タイムライン**：日々の記録を一覧表示。カテゴリ（楽天／Amazon／Yahoo／社内・その他）や進捗状況（未着手／進行中／保留／完了）でフィルタできます
- **枝分かれタスク**：あるタスクの下に、関連するサブタスクを紐づけて表示できます（開閉可能）
- **開催日・着手日・期日**：タスクごとに日付を記録し、期日超過は自動でハイライトされます
- **集計・振り返り**：前期（2〜7月）／後期（8〜1月）で自動集計。評価シートに貼り付けられるテキストをワンクリックでコピーできます
- **バックアップ／復元**：記録をJSONファイルとして書き出し・読み込みできます
- PC・スマホ両対応のレスポンシブレイアウト

## データについて

記録はすべて **ブラウザのlocalStorage** に保存されます。サーバーには一切送信されません。

- ブラウザのキャッシュを削除したり、別の端末・別のブラウザで開いたりすると記録は見られなくなります
- アプリ内の「💾 バックアップ」から定期的にJSONファイルへ書き出しておくことをおすすめします

## 使い方（ローカルで開く）

`index.html` をブラウザで直接開くだけで動作します。ビルドや依存関係のインストールは不要です。

## GitHub Pagesで公開する場合

1. このリポジトリをGitHubにpush
2. `Settings` → `Pages` を開く
3. `Source` で対象のブランチ（例：`main`）とルートフォルダ（`/`）を選択して保存
4. しばらくすると `https://<ユーザー名>.github.io/<リポジトリ名>/` で公開されます

## 技術構成

- HTML / CSS / vanilla JavaScriptのみ（フレームワーク・ビルドツール不要）
- フォントはGoogle Fonts（Noto Sans JP）をCDNから読み込み
